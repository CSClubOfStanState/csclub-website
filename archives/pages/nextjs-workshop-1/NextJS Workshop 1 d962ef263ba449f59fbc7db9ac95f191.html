First off, let‚Äôs look at the semantics of HTML. If we are new to HTML, simply know that the more of these elements you use, the easier it will be for others to find information on your page and how others can interact with your page.

For this workshop, we will be using the following elements: <header /> ,<nav />, <main />. For the sake of time, these are the only elements we will go over. The content within each of these elements will be stylized later. (To make look pretty)

  • Header (header) ‚Üí Introductory content like the name of the site or the owner‚Äôs information, for us this will serve as the main header for our page.
  • Navigation (nav) ‚Üí A major group of hyperlinks that can take us anywhere on the internet, including across our page, for us this will serve as redirect links.
  • Main (main) ‚Üí The main contents of the webpage, for us this is will include the content below our <nav />.

Now, don‚Äôt get your hopes up, we are not done just yet. All we need to do is declare the proper ids and class names for our CSS and we are in business. If you look above, each class name or id has the name of HTML tag that precedes it.
